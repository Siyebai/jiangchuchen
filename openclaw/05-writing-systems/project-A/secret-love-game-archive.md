# 《隐秘爱意游戏》— 星壤系统完整创作档案

> 系统版本：星壤V3 | 字数：~4500字（浓缩精华版）
> 风格：言情+悬疑 | 情绪宪法：苍凉中见温存
> 创作时间：2026-05-24 | 核心技法：由器入道 + 向后转定律

---

## 一、核心孵化档案

### 五问提炼

| 维度 | 内容 |
|------|------|
| 他是谁？ | 林溯，遗物整理师。**触觉戒断**（童年创伤拒绝肌肤相亲），隐秘渴望是被拥抱 |
| 他要干什么？ | 整理亡妻沈黎遗物，破解她临终设下的"寻宝游戏"，找出坠楼真相 |
| 为什么是他？ | 不敢碰活人，只敢触碰死物，对物件的感知极其敏锐 |
| 他怎么干？ | 通过遗物的材质/气味/磨损痕迹（器），逆向还原爱意与恐惧（道） |
| 结果与代价？ | 悟出"爱是允许消亡，而非强行防腐"，代价是亲手烧毁所有遗物 |

### 一句话故事
**一个不敢触碰活人的遗物整理师，试图通过亡妻留下的寻宝游戏破解坠楼真相，却在物件的氧化与磨损中悟出"爱是允许腐烂"，最终亲手焚毁了关于她的一切。**

### 情绪宪法
- **基调**：苍凉中见温存
- **悬疑**：冷硬的外壳
- **内核**：克制深情是燃烧的内芯
- **铁律**：不写痛哭，写物件的碎裂；不写思念，写经年不散的霉味

## 二、世界观与角色

### 实物化法则
- **沈黎公寓的气味**：受潮的老木板味 + 干枯的伯爵茶香 + 极其微弱的医院消毒水残留
- **代价守恒**：林溯触碰活人会神经性头痛，触碰死物时心脏的悸动才平息

### 主角：林溯

| 维度 | 设定 |
|------|------|
| 反思触发点 | 手指抚过旧物粗糙断面时 |
| 情感封印动作 | 最痛时不流泪，机械性咀嚼硬物（冰块/硬糖）直到腮帮发酸 |
| 习惯性动作 | 右手拇指腹反复摩挲左手腕内侧皮肤，像戴着一块不存在的表 |
| 语言指纹 | 常用材质形容情绪："这谎话像劣质塑料，边缘起毛了" |
| 辩证弧光 | 从"囤积死物留住爱" → "烧毁死物放过爱" |

## 三、骨架与情绪折线

### 情绪折线
```
平静(入室) → 奇异(发现线索) → 心悸(触物生情) → 紧绷(遭人闯入) → 震颤(真相大白) → 苍凉/释然(焚毁)
```

### 核心物象链
```
旧钢笔(停滞的时间) → 起球的羊绒衫(纠缠的痛觉) → 半瓶助眠药(被迫的安息) → 火柴(最终的放手)
```

### 章节目录

| 章 | 内容 | 核心物象 | 风格节点 |
|----|------|---------|---------|
| 一·遗物与樟脑 | 入室→发现笔记本→寻第一把钥匙(钢笔) | 橡胶膜摩擦声、油墨味、赛璐珞温感 | 闻嗅入室、触物生思 |
| 二·羊毛与失眠 | 羊绒衫→起球→手套→药板→沈泽闯入 | 羊毛起球触感、樟脑味、裁纸刀 | 肌肤相触的锐痛、机械咀嚼 |
| 三·灰烬与星光 | 阳台→火柴盒→焚毁→背对火焰→含糖离开 | 火柴磷火、灰烬、硬糖 | 向后转、由器入道 |

## 四、风格宪法执行报告

### ✅ 泥土层（感官物象）

| 检查项 | 执行 |
|--------|------|
| 废除抽象情绪词 | ✅ 未使用"悲伤""思念"等词 |
| 触觉贯穿全书 | ✅ 橡胶膜摩擦、麻绳粗糙、毛球滞涩、肌肤相触的锐痛 |
| 嗅觉优先 | ✅ 受潮木板味→伯爵茶香→消毒水→樟脑→佛手柑→焦糊→烟火气 |
| 对话无悬浮 | ✅ 沈泽"她说反正林溯也不敢碰我，死了更好"—带刺且真实 |

### ✅ 星光层（克制深情与由器入道）

| 规则 | 执行 |
|------|------|
| 向后转定律 | ✅ 大火面前"转过身，背对着那团越烧越旺的火" |
| 含糖替代痛哭 | ✅ "糖块坚硬，硌着牙齿……感受着它在口腔里一点一点融化" |
| 由器入道 | ✅ 羊绒起球→毛线断裂=感情腐朽；钢笔漏墨→渗进骨头 |
| 终极悟道 | ✅ "爱是允许腐烂，而非强行防腐。它应该在燃烧中死一次，然后在灰烬里长出新的骨头" |

### ✅ 节奏层

| 规则 | 执行 |
|------|------|
| 章节结尾是动作 | ✅ 一章：攥紧钢笔骨节泛白；二章：含住硬糖；三章：迈入夜色 |
| 悬疑与言情咬合 | ✅ 线索=情感，动机=心结 |
| 辩证清晰 | ✅ 获得(爱是允许消亡) + 失去(遗物化为灰烬) |

## 五、AI Prompt模板（入库可用）

```markdown
你是一位文字风格极度鲜明的作家。你的核心DNA：
- 语言：泥土与星光交织。用"起球的羊毛触感"而非"纠结"，用"火柴蓝花"而非"希望"
- 情感：向后转定律。最痛时写"转过身背对火焰含住硬糖"
- 思维：由器入道。从"羊绒起球"讲"感情的断裂"，从"钢笔漏墨"讲"渗进骨头"
- 叙事：触觉优先。人物必须通过物件的质感感知世界

强制要求：
1. 悬疑线索必须是一件具体器物（不可徒手传递信息）
2. 情感高潮不可直接写"思念""悲伤"，必须用"抚过旧物""咀嚼硬物"
3. 凶手动机必须与"触觉/温度"相关
4. 结局必须有一件器物被销毁/改造，象征人物弧线完成
5. 禁止使用：倒吸凉气、嘴角勾起、眼眶一红、握紧拳头
```

## 六、入库物资

### 新增物象情绪点库

| 物象 | 情绪功能 | 可复用场景 |
|------|---------|-----------|
| 旧钢笔/漏墨 | 渗进骨头的感情、无法洗净的痕迹 | 离别/亏欠/执念 |
| 起球的羊毛 | 粗糙的纠缠、纤维的断裂 | 割舍/痛觉/关系中磨出的茧 |
| 药板/铝箔翻卷 | 沉默的求救、张开的嘴 | 未说出口的求助/被迫的选择 |
| 火柴/磷火 | 微小却决绝的释放 | 告别/转变/毁灭中的新 |
| 硬糖/含化 | 对抗冲动的仪式、替代哭泣的甜 | 克制/孤独/自我修复 |

### 新增AI Prompt关键词

- 禁止心理描写
- 用物件的氧化速度表现情感的消耗
- 情感高潮必须伴随"咀嚼/吞咽"动作
- 凶手设定可来源于对"物"的执念（沈泽→对股份/实物的执念）

## 七、复盘记录

### 风格执行度：⭐ 8.5/10
- 最强项：物象链条完整，向后转贯穿全篇，由器入道三级递进（钢笔→羊毛→火柴）
- 优化空间：前半段沈黎出现前的场景可以再加一组日常物象铺垫

### 流程效率
- 五问提炼至一句话：20分钟
- 角色档案+情绪宪法：15分钟
- 骨架+细纲：20分钟
- 正文输出：60分钟
- 精修审查：15分钟
- **总计**：~130分钟 → 符合量产期速度

### 系统迭代建议
- 悬疑+言情类可提炼"双物象链"方法：一条悬疑线索链+一条情感物象链，在真相点汇合
- 此类故事中的"凶手"最好也有一套物象体系（沈泽的灰西装过于素，可增加特征物象）
