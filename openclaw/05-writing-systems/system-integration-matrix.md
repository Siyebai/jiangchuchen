## 核心差异：李白 vs 星壤

### 底层哲学完全不同

**李白说**：写人不是写故事。故事的灵魂不在结构，在人物的选择里。让一群角色活起来，然后他们自己去发生故事。

**星壤说**：创作是精密工程。由器入道——一切哲理从具体物件生发；向后转——最痛时写转过的背。

> 李白是"角色驱动"的人本主义，星壤是"物象驱动"的工程主义。

### 互补点

| 李白强项 | 星壤强项 |
|---------|---------|
| 短篇节奏控制（8000-30000字） | 长篇结构工程（成书级） |
| 角色差异化（每个人物独特声音） | **双重性档案**（更系统的角色建模） |
| 口语魂（像人说话的对话） | **语言指纹**（职业/地域感词汇系统） |
| 热叙事（允许直接写情绪） | **向后转定律**（更高级的情绪表达） |
| 感官具体化（基础版） | **五感沉浸**（精细版+嗅觉优先） |
| 三遍写作（三遍优化） | **六阶段门径**（更严谨的流程控制） |
| — | **AI协作Prompt**（量产加速） |
| — | **品控门径**（质量保障） |

### 建议融合策略

**写短篇（知乎向）** → 以李白系统为主干，借用星壤的：
- 向后转定律（替代李白的"情感外化"规则，它更精准）
- 语言指纹系统（替代李白的"口癖"规则，它更完整）
- 五感沉浸中的"嗅觉优先"（替代李白的"感官具体化"）
- 代价守恒原则（避免短篇情绪崩塌）

**写长篇（成书向）** → 以星壤系统为主干，借用李白的：
- 口语魂规则（避免星壤对话太"散文"）
- 反陈词滥调清单（星壤的泥土层审查可借鉴）
- 三遍写作的"冷置"节奏（星壤的门径可留出冷置时间）

## 合并后的统一写作规则集

### 一、风格宪法（两系统共识）

```
1️⃣ 感官先行：开篇第一句必须是环境感知，优先嗅觉/触觉
2️⃣ 情感外化（向后转）：不说"悲伤"，写"洗碗洗了四遍"
3️⃣ 物象生哲理（由器入道）：一切哲思从具体物件生发
4️⃣ 各人有各话（语言指纹）：对话带职业/地域/性格特征
5️⃣ 每章结尾必须是具体动作，非内心独白
6️⃣ 情绪高潮后必须有一个"风格沉淀点"（反思/物象感悟）
7️⃣ 禁止使用：不禁、竟然、倒吸凉气、嘴角勾起、捋胡子、拍桌子、眼眶一红
```

### 二、角色塑造统一模板

```
姓名：
一句话人设：
核心缺陷 → 隐秘渴望：
语言指纹（带职业感的口头禅）：
情感封印动作（最痛时做什么）：
习惯性物件动作（+什么物件）：
反思触发点（什么时候自我审视）：
辩证弧光（获得什么+失去什么）：
```

### 三、写作流程选择

```
目标 → 知乎8000-30000字短篇 → 走李白系统＋星壤点睛
目标 → 长篇成书 → 走星壤六阶段＋李白口语魂润色
目标 → 投稿初稿（不限长度）→ 星壤阶段四量产＋阶段五快速审计
```
