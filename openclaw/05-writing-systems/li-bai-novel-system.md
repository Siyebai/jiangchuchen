# 李白小说创作系统 · 完整版 v1.0

> 适用：8000-30000字知乎中短篇小说  
> 风格定位：**多角色差异化叙事 · 热叙事底 · 口语魂 · 反陈词滥调**  
> 核心原则：每个人物都有自己的声音；每一处描写都拒绝"似曾相识"。  
> 反AI铁律：不完美才是人，完美即是AI。陈词滥调是AI的温床，直觉的观察才是人的印记。

---

## 李白小说创作系统 总纲

### 写人，不是写故事

很多教程教你"怎么写出好故事结构"。  
但这套系统教你：**怎么让一群人活起来，然后他们自己去发生故事**。

故事的灵魂不在结构里，在人物的选择里。  
人物的真实感不在描述里，在声音和动作的差异里。  
作品的风格不是"作者一个人的声音"，而是**这群角色各自发声后混合成的交响乐**。  
**而这一切的前提是：拒绝陈词滥调。**

## PART ONE：风格系统（核心）

### 1.1 叙事风格定位

**基调：热叙事底 · 口语魂 · 反陈词滥调**

| 层级 | 定义 | 核心特征 |
|------|------|---------|
| **叙事底层** | 叙述者的"温度" | **热** —— 允许情绪直接书写，允许内心戏 |
| **对话层** | 角色说话的方式 | **口语魂** —— 像人说话，有口癖、有废话、有突然转移话题 |
| **动作层** | 角色行动的方式 | **差异化** —— 每个人物独特的动作习惯和反应模式 |
| **观察层** | 描述的角度 | **反陈词滥调** —— 不写"捋胡子"，写"手抬到一半又放下了" |

### 1.2 热叙事底层详解

**什么是"热叙事"：**
- 允许直接写"她感到一阵撕裂的痛"
- 允许写内心独白、情绪爆发
- 允许用比喻营造氛围

**但不等于"滥情"：**
- 热叙事 ≠ 每句话都煽情
- 热叙事 = 需要温度的时候，你有温度可用
- 高明的热叙事是**收放自如**——十个冷句里放一个热句，那个热句才打得动人

**热叙事的句式特征：**

```
【常用句式模板】
✓ 体感开头："胸口发闷，像压了块石头。" 
✓ 比喻递进："眼泪不是流下来的，是砸下来的。一颗一颗，砸在手背上，烫的。"
✓ 直接内心："她在心里骂了自己一万遍——不哭，不准哭。但眼泪不听她的。"
✓ 生理反应替代情绪："端着水杯的手在抖。水洒出来，烫红了虎口。她没感觉。" ← 热+暗
```

### 1.3 口语魂详解

**什么是"口语魂"：**
- 对话像人说的话，不是"戏剧台词推进剧情"
- 允许角色说废话、说重复的话、说前后矛盾的话
- 用口癖区分人物

**口语魂的三不放：**
1. 不说"你为什么要这样做" → 说"你认真的？"
2. 不说"她感到一阵心痛" → 说"疼。是真的疼那种。"
3. 不说"他沉默了片刻" → 说"他没说话。我也没催。"

**口语化对话检测标准：**
把对话读出来。如果读出来的感觉像在"演话剧"，删了重写。
如果读出来的感觉像"朋友在聊天"，通过了。

### 1.4 反陈词滥调系统

#### 什么是陈词滥调

陈词滥调 ≠ 错误。陈词滥调 = 别人写过的、读者看过很多遍的、缺少辨识度的描写。

**常见网文陈词滥调清单：**

| 陈词滥调 | 问题所在 |
|---------|---------|
| 老者捋了捋胡须 | 一万个作者写过的动作，读者看了一万遍 |
| 大婶双手叉腰 | 同上 |
| 大汉一拍桌子 | 同上 |
| 美人轻咬下唇 | 同上 |
| 他冷笑一声 | 同上 |
| 她红了眼眶 | 同上 |
| 他沉默了 | 同上（太模糊，没有画面）|
| 他眯起眼睛 | 同上 |

这些不是"错误"——读者不会因为出现这些而觉得你写得不好，但也不会因此记住你。记住你的唯一方法是：给出他们没看过的描述。

#### 反陈词滥调的三把武器

**武器一：负空间描述法**

不写"他做了什么"，写"他没做什么"。

| 陈词滥调 → 负空间描述 |
|---------|-----------|
| 他捋了捋胡须 → "他抬起手，指腹碰了碰下巴。没捋。就碰了一下，又放下了。" |
| 她红了眼眶 → "眼睛没红。但她眨了好几次眼，像眼睛里进了什么东西。" |
| 他拍了桌子 → "手抬起来，五指张开，悬在桌面上方。没拍下去。悬了三秒，又放下了。" |

**原理：** 读者预期中"应该出现的动作"没有出现——这种落空感会产生记忆点。

**武器二：异常细节法**

不描述"正常情况"，只描述"哪里不对劲"。

| 陈词滥调 → 异常细节 |
|---------|-----------|
| 街上有行人 → "街对面蹲着一条黄狗。尾巴僵着——没摇。说明它对这个陌生人感到不安。" |
| 她在哭泣 → "她没哭出声。但眼泪流到下巴的时候，她没有擦——让它挂着。这才是最难过的哭法。" |
| 他打量四周 → "他注意到墙上有一道裂缝。从上到下，笔直的，不像是自然的裂痕，像是被什么东西用力劈过。" |

**原理：** 正常的细节人人都会写。异常的细节需要观察者自己的眼睛。

**武器三：人物化描述法**

把物品、场景、天气当作有性格的东西来写——但不是拟人，而是赋予它们与剧情无关的态度。

| 陈词滥调 → 人物化描述 |
|---------|-----------|
| 墙上的匾写着... → "那块匾大概是上了年头了。'王氏铁器'四个字，'器'字少了一个口，漆皮剥了，像是一口缺了的牙。" |
| 风铃响了 → "远处有一座塔，每层檐角都挂着风铃。但有一层没响——大概是风铃掉了，或者锈住了。" |
| 照例的日落 → "太阳下山了，但巷子里还有一截墙是金黄色的。只有这一截——像一个舍不得走的人还在门口站着。" |

**原理：** 让这个世界有"自己的脾气"。读者不只看到场景，还感觉到场景在"表达"什么。

#### 反陈词滥调写作三问

每写完一段描写，停下来问自己：
1. 这个动作/描写，是我自己观察到的，还是我"记得别人这样写过"？
2. 如果我删掉这一段，会对故事有影响吗？
3. 这个描写，读者之前大概看过多少遍类似的？（0-10遍→通过，10-100遍→考虑换，100+遍→必须换）

#### 反陈词滥调对照表（速查版）

| 别写 | 写 |
|------|----|
| 捋胡子 | 手抬到下巴位置，碰了一下，又放下了 |
| 叉腰 | 手按在腰侧，拇指来回摩挲衣料 |
| 红了眼眶 | 没红。但眨眼的频率变快了。 |
| 眯起眼睛 | 眼睛没动。但嘴角收紧了。 |
| 冷笑 | 没笑。但鼻翼微微扩张了一下。 |
| 沉默了 | 他看着窗外。风从窗缝里挤进来，发出细小的哨声。 |
| 叹气 | 他吸了一口气，又慢慢地、无声地呼出来。 |
| 拍了桌子 | 手抬起来，五指悬空，没拍。 |

### 1.5 多角色差异化系统

#### 角色声音设计表（每个主要角色一张）

```
角色名：______
口癖/标志词：______
说话节奏：（慢/快/喜欢铺垫/喜欢直奔主题）
情绪表达方式：（直接说/用动作/用表情）
标志性小动作：______
紧张/说谎时的特点：______
句式偏好：______
观察世界的滤镜：______（赌徒看风险 / 老兵看安全 / 书生看逻辑 / 母亲看孩子）
```

#### 差异化三原则

1. 遮住角色名，看对话就知道是谁在说话
2. 角色说话方式不因剧情需要而改变（人设崩塌是大忌）
3. 副角色可以简单，但主要角色必须有≥2个独特标签

#### 同一件事，不同角色的反应

| 角色类型 | 同一场景的反应 |
|---------|--------------|
| 直性子 | "你这话什么意思？当着我的面说？" |
| 阴着来 | 笑了笑，端起茶喝了一口，没接话。 |
| 高情商 | "诶不至于不至于，她也不容易。"——转头传开。 |
| 赌徒（滤镜） | 他不说话。他在数桌上的指纹。 |

### 1.6 本系统写作的"体质"定位

```
【叙事体质】
底层温度：热（允许直接的情感描写）
切换能力：冷热自如（该收时收，该放时放）
语言质地：偏向口语（但保留文学性）
感官偏向：体感>视觉>听觉（多写"感受"少写"看到"）
情绪输出：直接（不绕弯子）

【角色哲学】
≠ 作者单声道的叙事
= 多声道混音的写作
≠ "我在讲一个好故事"
= "让这群人活起来，他们自己去发生故事"
```

## PART THREE：策划系统

### Stage 1：灵感孵化

**一句话模板：** [什么人]在[什么处境]下，做了一件[反常的事]

**三句话脑暴（角色版）：**
- 起点：她的状态是什么？她在乎什么？
- 转折：ta做了什么？说了什么？
- 终点：她变成了什么样子？

**淘汰机制：**
1. 冲突够强吗？
2. 有没有见过？
3. 我心里有感觉吗？

通不过就去下一题。不要硬写。

### Stage 2：人设塑魂

**角色设计卡：**
```

矛盾（他说的vs他做的）：______

```

**配角三元法：**
```

功能：______
说话特点：______
```

### Stage 3：雪花细纲

**第一层：** 一句话大纲（30字内）
**第二层：** 三幕结构（15%-65%-20%）
**第三层：** 8-15场戏，每场一句话核心
**第四层：** 字数分配

### Stage 4：导语钩子

**四大导语套路：** 开局即地狱 / 认知反转 / 疯批独白 / 死亡倒计时

**导语铁律：** 第一句话必须有冲突。三秒内知道"谁在干嘛"。留白比说透重要。

**导语=冲突+人物亮相：**
> ❌ "林深和白悦然的婚戒刺痛了她的眼睛。她决定分手。"
> ✅ "林深搂着白悦然进来的时候，我正端着手冲咖啡往阳台走。看见她那枚和我同款的戒指，我笑了一下，把咖啡倒进花盆里。花死了，我醒了。"

### Stage 5：情绪地图

**五层递进：** 背景共情 → 小事变大事 → 冲突延伸 → 鸟语花香 → 战神归来

**每一场三个情绪标记：** 场3：愤怒 → 委屈 → 冷静

## PART FOUR：三遍写作 · 三遍优化

### 第一遍

#### 4.1 第一遍写作（口语化初稿）

**核心方法：说故事，不写故事**——用嘴说，语音转文字。

写对话时，闭上眼睛，想象这个角色站在你面前。听到他说什么，你就写什么。不要修改。

**写完一段后自检：**
- 这段对话读出来像真人在说话吗？
- 遮住角色名，还能看出是谁在说话吗？
- 有没有哪句话太"完美"了？

#### 4.2 第一遍优化（语言质量深修）

**热叙事三箭：**
1. 情绪可感化：抽象情绪词→可感的画面/动作/体感
2. 感官优先：能用体感/触觉表达的，不用"看到"
3. 收放有度：十个冷句插一个热句

**口语化检测：**
- 把全文读一遍，标记"读着不像人话"的地方
- 对话有"废话"吗？（保留——废话才是人话）
- 叙述有"书面语"吗？（"他沉默了片刻"→"他没说话"）

**多角色差异化检测：**
- 每个主要角色至少在2个维度上不同：说话方式 / 行为模式 / 价值观
- 把每个角色的台词单独抽出来，遮名能否分辨？
- 警告：两个角色用了相似句式？情绪出口一样？冲突反应一样？

**反陈词滥调检测：**
- 有没有用了常见动作模板？→ 换负空间描述或异常细节
- 用反陈词滥调三问逐段过

#### 4.3 冷置沉淀（12-24h）

### 第二遍

#### 4.4 第二遍写作（重构版）

凭记忆重写，不翻第一遍原稿。

**导演心法：**
- 男主会站在哪？（站姿展现权力关系）
- 他手里在做什么？（转笔？摸后颈？）
- 他开口之前做了什么停顿？

**这一遍重点增加：**
1. 与剧情无关的小动作（每个角色2-3个）
2. 反逻辑的细节（全文2-3处）
3. 角色之间的"声音冲突"（对话重叠3-5处）

#### 4.5 第二遍优化（场景打磨）

- 场景检测：谁在？谁在什么位置？手里在做什么？
- 对话重叠检测：找2-3处加入打断/抢话
- 沉默的力量：找一处"该说话但没说话"
- 角色一致性检测：人设不崩塌

#### 4.6 冷置沉淀（6-12h）

### 第三遍

#### 4.7 第三遍写作（精修版）

逐段精修。

**精修四问：**
1. 这段的情绪是什么？
2. 这段有没有"废句"？
3. 这段的节奏对吗？
4. 这段之后读者对哪个角色理解更深了？

#### 4.8 第三遍优化（音韵与余味）

- 出声朗读全篇，标记读不顺的地方
- 僵尸词清除：就、那、很、非常、突然、然后、其实
- 句式节奏：连续5句一样长→穿插短的；平均句长15-20字
- 结尾余味：不要大团圆，要意犹未尽
- 差异化最终检测：再抽各角色台词，遮名辨识

### 4.9 完整案例：多角色差异化演示

**选题B「替身不回头」场景：男主发现女主真要搬走**

> 平庸版（所有角色说一样的话）：
> "你真的要走？""是的。""为什么？""累了。"

> 差异化版：
> 他站在门口，挡着她的行李箱。摸了摸后颈，张了张嘴，又闭上。
> "你认真的？"
> 她没抬头。把箱子上的灰擦了擦。"什么时候见我不认真过。"
> "那——"他停了一下，"三年？三年够不够？"
> 她终于抬起头，看他。"林深，你说三年前那句话的时候，也是这个表情。"
> 他摸后颈的手停住了。

## PART FIVE：写作质量保障体系

### 5.1 写作自查三问

1. 我在"创造一个角色让他自己说话"，还是"替角色写剧本"？
2. 我把所有角色都写成"我"的翻版了吗？
3. 这篇文章去掉作者名，能看得出是一个人的作品吗？

### 5.2 反AI准则

```
■ 禁止做的事：
   ❌ 写"完美"的句子
   ❌ 让所有角色说同一速度的话
   ❌ 回避"不完美"的表达

■ 必须做的事：
   ✓ 允许写"废话"
   ✓ 允许角色说前后矛盾的话
   ✓ 允许场景中有与剧情无关的元素

■ 写完后的自毁机制：
   如果你觉得"这句写得真好，太漂亮了"——删掉它。
   最美的句子一定是读者觉得"这句话就该是这个角色说的"。
```

### 5.3 金句检测

好金句的三种来源：
1. 角色说的话——不是作者想出来的，是角色必然说的
2. 反直觉的观察——"她哭不是因为难过，是因为他递纸巾的动作太熟练了"
3. 语言的节奏碰撞——长铺垫+短收尾

### 5.4 快速启动卡

```
□ 扫榜报告（当前热梗+趋势）
□ 选题确认
□ 灵感孵化（一句话→三句话脑暴）
□ 角色设计（每个主要角色填差异化表）
□ 雪花细纲（8-15场戏）
□ 导语钩子（冲突+人物亮相）
□ 情绪地图（5层+曲线）

□ 第一遍写作（口述→转文字→保留粗糙）
□ 第一遍优化 → 热叙事+口语化+差异化+反陈词滥调
□ 冷置12-24h
□ 第二遍写作（凭记忆重写→加小动作和反逻辑）
□ 第二遍优化 → 场景+对话重叠+沉默
□ 冷置6-12h
□ 第三遍写作（逐段精修）
□ 第三遍优化 → 音韵+余味+僵尸词清除
```

---

> **最终语：**
> 最好的小说不是"作者写了一个好故事"。
> 最好的小说是"作者创造了一群真实的人，然后他们有了一段难忘的经历"。
>
> 写作的本质不是"描述世界"，而是"用谁的眼睛看世界"。
> 赌徒有赌徒的眼睛，老者有老者的眼睛。
> 不用作者的眼睛替所有人看——让角色自己去看，世界自然不同。
> 不同的世界，自然不会"似曾相识"。
>
> 一切从"人"出发。写到每个人物都有自己的声音时，你就不需要担心AI味了。
> 因为AI只能制造"标准的范本"，而你在创造活人。
