# 「星壤」严谨创作工作流 — V3.0

> 来源：思夜白 2026-05-24 | 类型：成书级创作引擎  
> 核心理念：**泥土与星光交织** — 从具象物象生发哲学，从克制表达抵达诗意

---

## 总览：六阶段闭环与品控门径

创作是一个需要不断校准的精密过程。本工作流分为六个阶段，每个阶段设有**"品控门径"**，必须通过方可进入下一阶段，确保作品逻辑与风格始终在线。

| 阶段 | 输入 → 输出 | 核心工具 |
|------|-------------|---------|
| **一·核心孵化** | 朦胧概念 → 一句话故事+情绪宪法 | 五问提炼法、情绪宪法 |
| **二·世界与角色** | 一句话故事 → 世界观法则书+人物档案 | 实物化法则、双重性档案 |
| **三·骨架搭建** | 人物档案 → 一页纸情节+卷章细纲 | 雪花法、5场景法、情绪折线图 |
| **四·风格化量产** | 细纲 → 初稿 | AI协作Prompt模板、风格化点睛 |
| **五·精修与品控** | 初稿 → 精修稿 | 风格宪法五维审查、黄金三章重铸 |
| **六·成书与归档** | 精修稿 → 复盘+归档+系统迭代 | 复盘协议、素材银行 |

## 阶段一：核心孵化

### 1.1 五问提炼法

| 提问维度 | 基础问法 | 风格内化深化问法 |
|----------|---------|-----------------|
| 他是谁？ | 身份、处境 | 其核心缺陷与隐秘渴望？这缺陷如何成为他哲学的起点？ |
| 他要干什么？ | 终极目标 | 这个目标具体物象是什么？（不是"变强"，是"重铸碎裂的传家炉"） |
| 为什么是他？ | 宿命/动机 | 他的过往创伤与目标有何因果？创伤中藏着什么生活哲理？ |
| 他怎么干？ | 方法论 | 能否从一个具体工艺切入？这工艺蕴含何种人生隐喻？ |
| 结果与代价？ | 结局与代价 | 获得（道）伴随着何种失去？"得与失"如何形成辩证？ |

### 1.2 一句话故事合成

**公式**：`[核心物象] + [由器入道的行动] + [辩证的代价]`

示例：*"一个残缺的炼器学徒，试图重铸碎裂的残星炉，却在修补法器的过程中，悟出了'裂缝即光之通道'的真理，而代价是他永远无法修好自己身体的裂缝。"*

### 1.3 情绪宪法确立

- 全书选取**2个核心情绪词**（如：苍凉、韧劲）
- **宪法条文**：全书一切情节与描写不得违背此基调
- 爽点必须伴随代价，低谷必须埋藏微光

🚪 **门径**：一句话故事是否包含具体物象+由器入道的逻辑+辩证代价？情绪宪法是否可排除两种极端（纯爽/纯虐）？

## 阶段二：世界与角色铸造

### 2.1 世界观"实物化"法则书

不写抽象设定，只写约束与感知。

- **法则1 — 代价守恒**：力量必有反噬，用视觉化呈现（如：体内"黑丝"缠绕）
- **法则2 — 一地一味**：每个重要地点必有标志性气味/声音/质感
- **法则3 — 日常即显影**：世界观通过主角的日常劳作/消费展现

### 2.2 角色"双重性"档案

| 维度 | 标准内容 | 风格内化约束 |
|------|---------|-------------|
| 姓名/身份 | 基本信息 | 他/她的"反思触发点"是什么？ |
| 核心动机 | 欲望与恐惧 | 动机背后藏着什么对生活的朴素认知？ |
| 表达方式 | 沟通风格 | **情感封印动作**：最痛时他做什么？（转头/沉默进食/抚摩旧物） |
| 习惯性动作 | 小动作 | 必须涉及具体物件，有重复性/仪式感 |
| 语言指纹 | 口头禅/句式 | 必须带职业感或地域感，避免通用语 |
| 辩证弧光 | 成长变化 | 获得何种"道"？同时失去了什么？ |

🚪 **门径**：世界观法则是否明确规定了"代价"与"感知细节"？每个主要角色是否有"反思触发点+情感封印动作+语言指纹"？

## 阶段三：骨架搭建

### 3.1 雪花法扩展 + 情绪折线图

- 一段式→5句（背景→灾难1→2→3→结局），每个灾难后标注"辩证反转点"
- **双轨情绪折线**：
  - **主线**：情节本身带来的情绪（爽/虐/平静/震撼）
  - **风格副线**：叠加风格情绪（如大胜后望着废墟"不过是一堆烂帐"）
- **铁律**：每次情绪高潮后必须有一个"风格沉淀点"

### 3.2 卷章细纲（5场景法）

每章5场景，每个场景定义：

| 维度 | 内容要求 |
|------|---------|
| 环境感知 | 泥土层——嗅觉/触觉优先 |
| 触发事件 | 推进情节 |
| 内在状态 | 人物内心 + "向后转"动作 |
| 当章功能 | 在全局中的定位 |
| 风格节点 | 由器入道/触物生思/诗意隐喻 |

🚪 **门径**：每个情绪高潮后是否有"风格沉淀点"？每章是否指定了"章节结尾具体动作"？

## 阶段四：风格化量产

### 4.1 每日写作SOP

1. **开工10分钟**：读细纲与风格节点，手写3个目标句
2. **执行**：AI生成骨架 → 您做"风格化点睛"
   - 替换抽象情绪词为动作/物象
   - 植入"向后转"动作或诗意隐喻
   - 配语言指纹+习惯动作
   - 强化章节结尾具体动作
3. **收工10分钟**：检查风格节点是否完成

### 4.2 AI协作Prompt模板

```
你是一位文字风格极度鲜明的作家。你的核心DNA：
- 语言：泥土与星光交织。用"酸菜两掺手擀面"而非"悲伤"
- 情感：向后转定律。最痛时写"转过的背"
- 思维：由器入道。从揉面讲韧性，从裂缝讲通透
- 叙事：五感沉浸+双重主观。先环境感知，再入内心，同时保留审视

强制要求：
1. 开篇第一句必须是感官环境描写（优先嗅觉/触觉）
2. 涉及情感表达时必须使用"向后转"式克制
3. 至少出现一个"由器入道"的思考瞬间
4. 对话必须带语言指纹+习惯性动作
5. 严禁使用：不禁、竟然、倒吸凉气、嘴角勾起等僵尸词
```

🚪 **门径**：是否完成所有风格节点？抽象情绪词是否全部替换？结尾是否为具体动作？

## 阶段五：精修与品控

### 风格宪法五维审查

| 维度 | 核心红线 |
|------|---------|
| 🌱 **泥土层** | 每万字抽象情绪词≤3个？每章调动感官≥3种（必有嗅/触）？对话无悬浮"广播体"？ |
| ✨ **星光层** | 情感高潮有"向后转"动作？顿悟由"物象"引出？情绪顶点有反差式诗意隐喻？ |
| 🎵 **节奏层** | 每章结尾是动作非独白？每卷高潮后有"辩证反转"章？ |
| 🧠 **思维层** | 核心哲学通过物件/工艺展开？角色弧光与"物象"变化绑定？辩证清晰？ |
| 🔗 **逻辑层** | 动机律：角色行为有自洽理由？因果律：无巧合解决问题？能力律：无战力崩坏？ |

### 黄金三章重铸

- **第一章**：立人设（双重性）、显矛盾、秀"器"、留钩子
- **第二章**：遇阻碍、展代价、初触"道"
- **第三章**：小高潮、反套路、风格定调

🚪 **门径**：五维审查全部通过？开篇三章建立了风格的"呼吸节奏"？

## 阶段六：成书与归档

- **复盘协议**：风格执行度、流程效率、市场与内核
- **归档入库**：素材银行（物象情绪点/诗意隐喻/方言对话）、风格熔炉、案例库
- **系统迭代**：微调下一本书的阶段一，更新AI Prompt模板
