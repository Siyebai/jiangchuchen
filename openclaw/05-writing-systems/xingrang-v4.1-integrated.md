## 前置引擎A：热榜扫查（来自李白Stage 0）

每次创作前必做，输出本期创作方向。

### 扫什么

| 数据源 | 重点 |
|--------|------|
| **知乎盐选热榜** | 前50篇标题/导语/分类/字数 |
| **小红书"知乎写作"搜索** | 编辑推荐例文、爆款拆文 |
| **小红书热榜"短篇小说"** | 高赞笔记趋势方向 |
| **编辑朋友圈/群** | 最近收什么、拒什么 |

### 扫榜三步法（30分钟）

```
Step 1 数据采集(10min) → 爬前50标题/分类，整理到表
Step 2 模式识别(10min) → 找重复梗、新梗、高互动但未见的类型
Step 3 趋势判断(10min) → 什么上升？什么饱和？切入点在哪？
↓ 输出
【本期创作方向】：______
【热梗排行】：1.___ 2.___ 3.___
【差异化切入点】：______
```

### 拆文复盘→数据回馈（李白复盘系统）

每次完稿或读到爆款后：

```
① 拆解爆款标题/导语结构
② 记录其热梗组合方式
③ 标记值得学习的技术点
④ 归入扫榜数据库 → 下期参考
```

## 前置引擎B：一步到位通道（来自李白核心）

当时间紧迫或灵感已成熟时，可不走完整六阶，通过此通道直达量产。

### B1 一句话灵感快速孵化

在进入星壤"物象开口·哲思寄生"之前，先用李白的方式快速锁定故事核心：

```
公式：[身份 + 处境] + [反常行为] + [一句话冲突]

例如："被贬冷宫的贤妃重生成选秀女当天，当众拒了皇上。"
```

### B2 快速人设塑魂（三要素框架）

在进入星壤「质感锚点档案」前，先用李白三要素打底：

| 要素 | 李白问法 | 示例（林溯） |
|------|---------|------------|
| **欲望** | 角色想要什么？ | 想被拥抱而不头痛 |
| **恐惧** | 角色害怕失去什么？ | 害怕触碰后再次被推开 |
| **缺陷** | 性格/能力弱点 | 触觉戒断——不敢碰活人 |

> 这步只花3分钟，为后续星壤"质感锚点"提供靶心。

### B3 导语钩子工坊（李白Stage 4精华）

在正式写作前，先打磨一个300字以内的导语钩子。

```
黄金公式：
[反常细节] + [情感冲击] + [悬念收束]

示例（隐秘爱意游戏）：
"钥匙插入锁孔时，林溯闻到了那股味道。
不是死寂房间的霉味——是受潮的老木板、干枯的伯爵茶香、
和极其微弱的消毒水残留。
沈黎死了三十七天。但他一进门就知道——
这间屋子里，进来过别人。"
```

**导语钩子自检清单：**
- 第一句是否抓人？（反常/悬念/感官冲击）
- 前100字是否建立了画面？
- 结尾是否埋了一个钩子？

## 六阶算法流（星壤V4主体 + 李白模块注入）

### 🚀 阶段一：哲学寄生孵化

**核心保持不变（星壤V4原生）**。在完成"物象开口·哲思寄生"后，增加一分钟快速验证：

**李白注入：灵感验证三问**
```
① 这句话我能用一句话讲给别人听，让人想追问"然后呢"？
② 我的物象（钢笔/胶片/相纸）是否有三种以上的感官特质？
③ 我的主角在三句话内是否让人感觉到"ta有故事"？
```

---

### 🚀 阶段二：质感赋魂铸造

**核心保持星壤V4的「质感锚点档案」**。在填写质感锚点之前，先走李白注入：

**李白注入①：人设快速塑魂（3分钟）**

| 维度 | 李白快问 | 填写 |
|------|---------|------|
| 欲望 | 角色最想要什么？用一个具体场景写出来 | |
| 恐惧 | 角色最怕发生什么？用一句台词写出来 | |
| 缺陷 | 这个缺陷如何推动情节？ | |

**李白注入②：口语魂规则（对话层注入）**

在完成质感锚点的「语言指纹」后，额外绑定三条口语魂规则：

```
口语魂三不放：
1. 不说"你为什么要这样做" → 说"你认真的？"
2. 不说"她感到一阵心痛" → 说"疼。是真的疼那种。"
3. 不说"他沉默了片刻" → 说"他没说话。我也没催。"
```

**李白注入③：反陈词滥调初始清单**

在角色档案底部，打上反陈词滥调禁止标签：

```
🚫 禁止：捋胡子、叉腰、拍桌子、吸鼻涕、倒吸凉气
🚫 禁止：嘴角勾起一抹、眼眶一红、握紧拳头
🚫 禁止：不禁、竟然、忍不住
🚫 禁止："她感到……""他心想……" 直述情绪
```

### 🚀 阶段三：算法生骨搭建

**核心保持星壤V4的「风格节点生成器」**。在构图时增加：

**李白注入①：导语钩子定点**

在第一章细纲顶部添加钩子占位符，写入前置引擎B3产出的导语钩子。要求：
- 第一章第一段必须完成钩子使命
- 钩子中必须出现至少一个核心物象

**李白注入②：情绪地图双轨制（补充说明）**

星壤已有双轨情绪折线（主线+物象衰变），在此基础上补充李白的热叙事温度标注：

```
在情绪折线上每个节点标注"热/冷"：
🔥 热 — 允许直接写情绪、内心戏、比喻抒情
❄️ 冷 — 克制、动作代替情绪、向后转

铁律：十个冷句里放一个热句。热句才打得动人。
```

---

### 🚀 阶段四：分层织造量产

**核心保持星壤V4的三层插件（泥土/星光/由器入道）**。在量产节奏中注入：

**李白注入①：冷置隔板**

在每遍写作之间插入冷置：
```
第一遍写作（口语化·快速）→ 🧊 冷置2-4小时 → 第一遍优化
第二遍写作（重构版）     → 🧊 冷置2-4小时 → 第二遍优化
第三遍写作（精修版）     → 第三遍优化 → 定稿
```

冷置期间做什么：不看稿。散步/阅读/写别的。让大脑忘记细节。

**李白注入②：口语魂润色插件**

在完成三层插件精控之后，追加一轮口语魂润色：

```
指令：扫描全文对话段落。
执行：
1. 检查每句对话是否像人说的话——有没有口癖？有没有废话？有没有突然转移话题？
2. 凡是"教科书式对白"（QA式问答），强制打散重组。
3. 每段对话必须伴随一个动作（配合质感锚点的习惯性动作）。
4. 禁止所有解释性旁白（"他解释道""她愤怒地回应"→直接用动作+语气体现）。
```

### 🚀 阶段五：全息透视品控

**核心保持星壤V4的品控看板（五感雷达图、抽象词热力图、收束饼图、入道路径、逻辑三律）**。在品控清单中注入：

**李白注入：反陈词滥调审查清单（精修最后一步）**

```
📋 反陈词滥调逐项排查：

□ 每千字是否有"不禁""竟然""忍不住"等虚词？若有→删除
□ 是否有"捋胡子""拍桌子""叉腰""背过身去"等通用动作？若有→替换为独有动作
□ 是否存在"倒吸一口凉气""嘴角勾起一抹笑"等僵尸表情？若有→删除
□ 每章是否有重复使用两次以上的比喻/意象？若有→替换其一
□ 对话中是否有"说明书式"介绍（如"我叫XX，是XX，来自XX"）？若有→打散融入

全部❌ → 通过门径
```
